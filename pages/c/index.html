C 端产品设计

主题

发现身边的美好（照片分享）

主要功能：

主体功能类似 Instagram，但有几点显著差异：

  1. 强调地理位置信息：每张照片必须对应一个线下商户的 beacon
  2. 使用微博账号登录
  3. 返利：用户 A 喜欢了 B 的照片后一定时间内去了该照片对应的商户，系统会给 B 返利
  4. 类似微博的用户关系：单向好友和双向好友（好友圈）。发照片时可以选择可见范围

1. 发照片

用户发表照片时需要和一个 Beacon 绑定（系统会显示用户遇到过的 Beacon 列表），支持滤镜，可以选择可见范围。 其它用户可以自由地 like 和评论，但遇到过该 Beacon 的人的评论会以特殊标记显示。

文案引导：引导用户只能发表美好的照片（积极向上的，不含抱怨的）。

2. 查看关注的人的照片

同微博一样可以切换自己关注的人和好友圈。列表中比较重要的信息：图、作者、like 和评论数量、地点、时间和评论摘要。

3. 发现

可以看到附近一定距离内的所有照片，同时提供类似陌陌的“漫游”功能——用户可以自由查看指定点附近的照片。

4. 个人资料

包含：

  1. 个人基本信息
  2. 自己的相册
  3. 到过的地点（地图形式）
  4. 自己得到的返利
  5. 通过微博关系和上传通讯录来找寻好友
